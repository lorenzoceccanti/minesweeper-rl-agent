\section{Related Work}

\begin{frame}{Related Work}
  \textcite{wang2025minesweepercnn} train a CNN-based DQN agent on
  Minesweeper and propose:
  \begin{itemize}
    \item a \alert{one-hot board encoding} (10 channels: values $0$–$8$
      plus one channel for unrevealed cells) fed to a convolutional
      $Q$-network;
    \item a \alert{classification-style reward} separating win, loss,
      informed progress, and blind guesses.
  \end{itemize}
  \begin{highlightbox}{Scope of the comparison}
    Statistical significance in this project targets the DRL techniques
    \emph{we} implement on our own environment, not a benchmark against
    external state-of-the-art results.
  \end{highlightbox}
\end{frame}
